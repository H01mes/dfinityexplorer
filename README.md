# DFINITY Explorer

[DFINITY Explorer](https://www.dfinityexplorer.org/) is an open-source block explorer for the DFINITY blockchain.

## Open-Source Libraries Used
* Bootstrap
* jQuery
* Popper
* Google Fonts
* Font Awesome
